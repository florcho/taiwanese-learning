# 🎙️ Plaud Note AI 프롬프트 모음

> Plaud 앱에서 녹음 후 → AI 분석 단계에서 아래 프롬프트들을 그대로 사용.
> 이 파일은 폰에서 빠르게 열어보기 위한 백업용. 앱에서 탭 한 번으로 복사도 가능.

---

## 1️⃣ Reels/영상 트랜스크립트 분석

**언제**: 인플루언서 Reels를 영상으로 보거나 Plaud로 녹음한 직후

**프롬프트**:
```
請將以下台灣中文逐句拆解，每句提供：
1) 原文（繁體中文）
2) 注音符號
3) 拼音
4) 韓文翻譯
5) HSK 4 以上的生詞，列出該詞的詞性與 2 個例句
6) 美容／醫美／網紅產業相關專業詞彙，請特別標註

請以 JSON 格式輸出，方便我直接複製到學習 app 裡。

[在這裡貼上轉錄文字]
```

**활용**: 결과 JSON을 `data/lessons.js`에 새 레슨으로 추가

## 2️⃣ Yuna 회의 답변 코칭 (가장 중요!)

**언제**: Yuna와의 미팅/통화/Slack 음성 메시지 직후

**프롬프트**:
```
請分析以下這段我與台灣同事 Yuna 的對話：

1) 將 Yuna 說過的每段話按時間順序列出，並提供繁體中文 + 韓文翻譯
2) 找出我需要回應的部分，為每一個提供 3 種等級的回應範例：
   - 簡單版（A2 程度）
   - 中等版（B1 程度，加入台灣常用口語）
   - 自然版（B2 程度，像本地人）
3) 列出 Yuna 使用的台灣特有口語（例如 "本來也想說..."、"不然就..."），並標註其用法

[在這裡貼上轉錄文字]
```

**활용**: 자연스러운 답변을 외워서 다음 미팅에서 활용

## 3️⃣ 이메일 분석 → 답장 초안

**언제**: 협력사 이메일 받자마자

**프롬프트**:
```
請分析以下這封台灣商業合作來信：

1) 重點摘要（韓文）
2) 對方在要求什麼？需要我回覆什麼？
3) 列出 5 個信中的商業用語＋韓文解釋
4) 提供 3 個版本的回信草稿（繁體中文）：
   - 接受版本
   - 婉拒版本
   - 需要更多資訊版本
5) 信尾的標準客套話有哪些？

[在這裡貼上信件內容]
```

## 4️⃣ 발음 교정 (내 녹음 vs 원본)

**언제**: 쉐도잉 연습 후, 본인 발음 확인하고 싶을 때

**프롬프트**:
```
我正在學習台灣中文。我會提供兩段錄音：
A) 原文／原始錄音（台灣人的發音）
B) 我跟讀的錄音（韓國人，剛開始學）

請：
1) 比對 A 跟 B 的發音差異
2) 找出我發錯的字、聲調錯誤的字
3) 重點挑 3 個最需要練習的發音點
4) 給我練習建議（韓國人常犯的台灣中文發音問題）

[附上錄音檔]
```

## 5️⃣ 격식 → 회화체 변환 (학습 효율 ↑↑)

**언제**: 격식 이메일 표현이 머리에만 들어가고 입에서 안 나올 때

**프롬프트**:
```
請將以下格式化的商業中文改寫成兩個版本：
1) 一個更口語的版本（像跟同事聊天）
2) 一個更正式的版本（像跟客戶寫信）

並列出每個版本使用的關鍵句型差異，例如 "由於" vs "因為"，"完畢" vs "完了" 等等。

[在這裡貼上原文]
```

## 💡 추가 팁

- Plaud 앱에서 트랜스크립트는 **繁體** 옵션 켜기 (간체로 떨어지면 학습 효율 ↓)
- 녹음 시 Yuna에게 미리 양해 구하는 게 좋음: "幫助學習用，方便錄音嗎？"
- TXT 파일로 export 하면 인코딩 사고 자주 발생 → **앱 안에서 텍스트 복사**가 가장 안전

---

## 🔧 인코딩 사고 시 응급처치

만약 Plaud TXT 파일이 깨졌다면 (예: "è¼ªä¸å" 같은 글자), 다음 Python 코드로 복구 시도:

```python
with open('broken.txt', 'r', encoding='utf-8') as f:
    text = f.read()
fixed_bytes = bytearray()
for ch in text:
    code = ord(ch)
    if code < 256:
        fixed_bytes.append(code)
print(fixed_bytes.decode('utf-8', errors='replace'))
```

다만 데이터 손실이 있을 수 있으니, **가장 좋은 방법은 Plaud 앱에서 직접 복사**.
